# Sample Audio Files

This directory contains sample audio files for testing the Clinical Assistant.

## Included Sample

- `sample-consultation.txt` - A text transcript you can use with the "Generate Audio" feature in the app to create a test audio file.

## Using the Generate Audio Feature

1. Start the application (`npm run dev`)
2. Sign in to your account
3. Click **"Generate Audio"** in the sidebar
4. Paste the sample transcript or write your own
5. Click **"Generate"** to create an audio file using Amazon Polly
6. Use the generated audio to test the transcription pipeline

## Alternative: Record Your Own

You can also use the **Record** feature in "New Conversation" to record a simulated doctor-patient conversation directly in the browser.
